Welkom op mijn oefenpagina

Dit is een voorbeeld van een eenvoudige webpagina waarin we enkele basiselementen gebruiken. De opmaak gebeurt volledig met CSS, zodat de structuur en de vormgeving netjes gescheiden blijven.

CSS maakt het mogelijk om de stijl van een website snel en overzichtelijk te wijzigen. Met één wijziging in een extern bestand pas je meteen de look aan van alle pagina’s die eraan gekoppeld zijn.

  • HTML structuur
  • CSS opmaak
  • Interactiviteit met JavaScript
